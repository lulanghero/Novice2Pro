# Novice2Pro
从新手到专业开源贡献者的成长助手

## 1. 项目简介

**Novice2Pror** 是一个面向新手与成长型开发者的开源教育与激励智能体平台。  
项目基于 GitHub 行为日志与 OpenDigger 指标数据，通过 AI Agent 技术，为开发者提供：

- 开源项目与任务推荐
  
- 数据驱动的学习路径规划
  
- 可解释的激励与成长反馈
  

---

## 2. 赛题与数据来源

- 赛事：OpenSODA 开源数字生态分析与应用创新大赛
  
- 赛题方向：开源教育与开源激励
  
- 数据来源：
  
  - GitHub 2020 年 1 月全域日志
    
  - GitHub Top300 仓库历史日志（2020–2023）
    
  - OpenDigger 指标数据（含 OpenRank）
    

---

## 3. 核心问题定义

当前开源新手普遍存在：

- 不知道从哪些项目开始
  
- 无法判断项目是否“友好”
  
- 贡献反馈慢，容易放弃
  
- 缺乏系统化成长路径
  

本项目旨在通过数据与智能体技术解决上述问题。

---

## 4. 数据方法论（字段—问题—功能）

| 字段（示例） | 所回答的问题 | 支撑功能 |
| --- | --- | --- |
| Issue 创建频率 | 新手是否容易参与 | 项目门槛评估 |
| PR 合并周期 | 社区反馈是否及时 | 项目健康度判断 |
| 贡献事件类型 | 用户处于哪个阶段 | 学习路径规划 |
| OpenRank | 贡献是否被认可 | 激励与等级体系 |

---

## 5. 技术路线

```
      数据层
GitHub 日志 + OpenRank
        ↓
      分析层
行为统计 / 时间序列 / 图分析
        ↓
      方法层
路径生成 / 推荐策略 / 激励模型
        ↓
      AI Agent 层
推荐 Agent / 规划 Agent / 激励 Agent
        ↓
      应用层
开源教育与激励平台
```
